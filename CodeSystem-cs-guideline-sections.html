CodeSystem: Guideline Sections

Official URL: http://fhir.awmf.org/awmf.ig/CodeSystem/cs-guideline-sections Version: 0.1.0
Active as of 2025-08-06 Computable Name: GuidelineSections

Sections of a guideline, describing various aspects such as its intention, target patient group, care setting, and other relevant contextual information.

This Code system is referenced in the content logical definition of the following value sets:

  • ContentTypesVS
  • GuidelineSectionsVS

This case-insensitive code system http://fhir.awmf.org/awmf.ig/CodeSystem/cs-guideline-sections defines the following codes:
